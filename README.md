# dbt-automation
Scripts for dbt automation

Example usage:

1. 

    python scaffolddbt.py --project-dir workspace/lahi \
                          --project-name lahi 
2. 

    python syncsources.py --project-dir workspace/lahi \
                          --source-name lahi 

3. 

    python flattenairbyte.py  --project-dir workspace/lahi
